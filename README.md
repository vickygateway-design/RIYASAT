# Rajasthani Dresses E-Commerce Website

## Overview
This is a comprehensive documentation for the Rajasthani Dresses e-commerce website, showcasing traditional Rajasthani dresses along with functionalities for users to browse and purchase.

## Features
- **Wide Range of Dresses**: Explore various traditional Rajasthani dresses, including sarees, lehengas, and more.
- **User Accounts**: Users can register and log in to manage their accounts and orders.
- **Shopping Cart**: A simple and intuitive shopping cart system for easy purchases.
- **Responsive Design**: Optimized for both desktop and mobile users.
- **Search Functionality**: Easy search options to find specific dresses.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/vickygateway-design/Riyasat.git
   cd Riyasat
   ```
2. **Install Dependencies**:
   Make sure you have Node.js and npm installed, then run:
   ```bash
   npm install
   ```
3. **Start the Development Server**:
   ```bash
   npm start
   ```
4. **Open Your Browser**:
   Navigate to `http://localhost:3000` to view the application.

## Customization Guide
- To customize the dress collection, modify the **dresses.json** file in the `data` directory with your dress details.
- Update styles in the `styles.css` file located under the `css` directory for changing the appearance of the website.

## GitHub Pages Deployment Instructions
1. **Update your GitHub Repository Settings**:
   - Navigate to `Settings` > `Pages` in your GitHub repository.
2. **Select the Branch for Publishing**:
   - Choose `main` or the branch you want to deploy from.
3. **Choose a Folder to Deploy**:
   - Select `/ (root)` if your entry point is in the root directory.
4. **Save and Publish**:
   - Click on `Save`. Your site will be published at `https://<username>.github.io/Riyasat` once the build process is complete.

## Conclusion
The Rajasthani Dresses e-commerce website provides a unique platform for users to explore and purchase traditional dresses. Follow the above instructions for setup, customization, and deployment.
